---

## 怎么做出来的

1. **数据层**：用 OCR 提取 CSSCI PDF 期刊目录 → 结构化 JSON（271 种）
2. **规范层**：从学术 skill 仓库 + 投稿须知 PDF 抽取 32 种期刊的完整注释体例（剩余 248 种用学科惯例推断）
3. **引擎层**：Python 实现引文解析器（17 条真实论文测试 100% 解析准确率）+ TypeScript 重写用于 Web 端
4. **接口层**：Next.js 14 App Router + Auth.js v5 + JSON 存储（dev）/ Postgres（prod）
5. **前端层**：Tailwind + shadcn 风格组件，4 个页面 + 2 个 API

整个项目从立项到 MVP 上线用了 **5 天**。
