# 引用通 / CiteTong — 产品需求（PRD · 1 页）

> 2026-06-16 v1 · MVP 范围 · 仅供 5 天冲刺用

## 1. 产品定义（一句话）

**投稿前最后一公里**：学者上传手稿 → 工具按目标期刊的注释体例自动校验注释 + 推荐目标期刊 + 给出改稿建议。

## 2. 目标用户

| 角色 | 占比 | 痛点 |
|------|------|------|
| **博士生**（主要） | 50% | 导师不细查注释体例，投稿前不自信 |
| **青年教师/博后** | 30% | 同时投 2-3 个期刊，每个体例不同 |
| **资深学者** | 15% | 偶尔投跨学科期刊，遗忘某刊体例 |
| **学术编辑/学生助理** | 5% | 帮导师/期刊做初审 |

## 3. 核心功能（MVP 必须）

| 功能 | 优先级 | 描述 |
|------|:------:|------|
| **F1. 引文解析** | P0 | 上传 Word/PDF/纯文本 → 自动抽取所有引文条目（含页下注、夹注、尾注） |
| **F2. 格式校验** | P0 | 按目标期刊（如《历史研究》《哲学研究》）体例逐条检查 → 输出 ✓/✗ + 错因 + 建议 |
| **F3. 期刊推荐** | P0 | 给论文摘要 + 关键词 → 按学科 + 引用偏好（权威性/速度/录用率）推荐 5 本目标期刊 |
| **F4. 改稿建议** | P1 | 把整段引用一键改成目标期刊体例（diff 显示） |
| **F5. 用户/订阅** | P1 | 注册/登录/按月订阅/用量统计 |
| **F6. 批量处理** | P2 | 一次性校验 50 条引文，导出 Word 报告 |

## 4. 不做清单（明示不做，避免范围蔓延）

| 不做 | 理由 |
|------|------|
| ❌ 查重/学术不端检测 | 知网/万方已有，赛道不同 |
| ❌ 全文翻译 | DeepL/ChatGPT 通用场景，非垂直需求 |
| ❌ 期刊官网投稿对接 | 各刊投稿系统封闭，无 API；改人工复制 |
| ❌ 文献综述自动生成 | 学术伦理雷区，不碰 |
| ❌ 投稿信/cover letter 自动写 | 锦上添花，非 MVP 必需 |
| ❌ 团队协作 | 学者单兵作战，初期无需 |

## 5. 用户旅程（核心 5 步）

```
┌─────────┐   ┌──────────┐   ┌─────────┐   ┌────────┐   ┌────────┐
│ 注册/登录 │ → │ 上传手稿  │ → │ 选目标刊 │ → │ 校验报告 │ → │ 一键改稿 │
│ (1元试用)│   │ Word/PDF │   │ (5 推荐)│   │ ✓/✗    │   │ 导出Word│
└─────────┘   └──────────┘   └─────────┘   └────────┘   └────────┘
```

## 6. 定价（阶梯）

| 套餐 | 月费 | 配额 | 目标 |
|------|------|------|------|
| **体验** | ¥1/7 天 | 5 篇 | 引流（公众号软文落地页） |
| **学生** | ¥39/月 | 30 篇 | 在读硕博（验证 ID） |
| **学者** | ¥99/月 | 100 篇 | 青年教师/博后 |
| **机构** | ¥399/月 | 500 篇 + 5 席位 | 课题组/编辑部 |

**年付 8 折 / 推荐返 ¥20 / 团购 5 人起 9 折**。

## 7. 成功指标（首月）

| 指标 | 目标 |
|------|------|
| 注册转化率 | 公众号软文 → 注册 ≥ 8% |
| 付费转化率 | 注册 → 付费 ≥ 5% |
| 留存（次月） | ≥ 40% |
| 单用户月均使用 | ≥ 3 篇 |
| 引流源 | 公众号 + 14 篇学术论文 + 小红书/知乎软文 |

## 8. 假设与风险

| 假设 | 验证方式 |
|------|---------|
| 学者愿意为注释校验付费 | 公众号发文 A/B 测试 ¥1 vs ¥0 首单 |
| 23 种已规范足够做种子 | D2 跑通 10 篇真实论文后再扩 |
| 公众号+14 篇论文能引流 | D5 软文上线后看 7 天数据 |
| 风险 | 缓解 |
|------|------|
| 格式规范采集成本高 | LLM 半自动：把《XX 期刊投稿须知》PDF → 解析 → 人工 review |
| 微信支付接入复杂 | D4 用 Creem/Stripe Checkout（中国大陆主体单独处理） |
| 引文解析误差大 | 提供手动标记 + 半自动校对界面 |

## 9. 5 天里程碑（详 → `ROADMAP.md`）

- **D2**：核心引擎（CLI demo 可跑）
